シンプルで効率的なGoレート制限ライブラリ

rRateLimitは、APIやサービスのリクエストレートを制御するための軽量なGoライブラリです。

使い方

インストール

go get github.com/rRateLimit/rRateLimit

基本的な使用例

package main

import (
    "fmt"
    "time"
    "github.com/rRateLimit/rRateLimit"
)

func main() {
    // 1秒間に3リクエストまで許可
    rl := rRateLimit.NewRateLimit(3, time.Second)
    
    for i := 0; i < 10; i++ {
        if rl.AllowRequest() {
            fmt.Printf("Request %d allowed\n", i+1)
        } else {
            fmt.Printf("Request %d denied\n", i+1)
        }
        time.Sleep(200 * time.Millisecond)
    }
}

API リファレンス

NewRateLimit(limit int, interval time.Duration)

新しいレート制限インスタンスを作成します。

  • limit: 指定期間内に許可されるリクエスト数
  • interval: リクエスト制限がリセットされる期間

AllowRequest() bool

リクエストが許可されるかどうかを確認します。許可される場合はtrueを返し、トークンを1つ消費します。
